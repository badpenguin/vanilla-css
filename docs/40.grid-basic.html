Responsive Grid

Cell responsive uses one of the responsive classes.

There are 5 default breakpoints: sm, md, lg, xlg, xxlg; its easy to add more.

The default grid has 2, 3, 4 and 5 column but you can use the mixin to build other as you wish.

For example: @include cell-builder(12, 's');

Cell at small size will always try to be single column.

In this example the "md" size is used.
