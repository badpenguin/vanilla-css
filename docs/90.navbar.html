Navbar without javascript

Try to resize to 1000px to have the hambuger menù appears. It works only onto the top navbar.

How It Works

The main container is: nav.navbar pluse a toggle class: navbar--toggle-on-medium

The modifier: .navbar--wrap allow for longer navbar to wrap instead of generating x-overflow.

The modifier: .navbar--toggle-always prevents the navbar wrapping and always show the toggle menù

The child is: ul.navbar__menu plus a position class: navbar__menu--center, left or right

The toggle button is not within the UL. Its into a separate element: div.navbar__toggle

The toggle can contain a LABEL, a CHECKBOX and the hidden overlay: div.navbar__overlay

It has an utility modifier: .navbar__overlay--right

And a close button with: div.navbar__overlay__close
